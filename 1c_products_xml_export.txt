ИНСТРУКЦИЯ: Экспорт товаров из 1С в XML для сайта
====================================================

1. Задача
---------
Сформировать XML-файл с товарами (id, название, цена, количество, картинка) и сделать его доступным по HTTP для интеграции с сайтом.

2. Пример структуры XML
-----------------------
<Products>
  <Product>
    <Id>1001</Id>
    <Name>Котел отопления BAXI</Name>
    <Price>35000</Price>
    <Quantity>12</Quantity>
    <ImageUrl>https://ваш_домен.ru/images/baxi.jpg</ImageUrl>
  </Product>
  <Product>
    <Id>1002</Id>
    <Name>Радиатор алюминиевый RIFAR</Name>
    <Price>4200</Price>
    <Quantity>34</Quantity>
    <ImageUrl>https://ваш_домен.ru/images/rifar.jpg</ImageUrl>
  </Product>
  <!-- ... -->
</Products>

3. Как сделать в 1С:Предприятие (пример для 1С 8.x)
----------------------------------------------------

A. Создайте внешнюю обработку (epf) или модуль в конфигураторе.

B. Пример кода для формирования XML:

Процедура СформироватьИВыгрузитьXML(ПутьКФайлу)
    // Получаем товары
    Запрос = Новый Запрос;
    Запрос.Текст =
        "ВЫБРАТЬ
        |   Номенклатура.Ссылка КАК Id,
        |   Номенклатура.Наименование КАК Name,
        |   Номенклатура.Цена КАК Price,
        |   Номенклатура.Остаток КАК Quantity,
        |   Номенклатура.Картинка КАК ImageUrl
        |ИЗ
        |   Справочник.Номенклатура КАК Номенклатура";
    Результат = Запрос.Выполнить();
    Выборка = Результат.Выбрать();

    XMLДок = Новый XMLWriter;
    XMLДок.УстановитьСтроку();
    XMLДок.ОткрытьЭлемент("Products");

    Пока Выборка.Следующий() Цикл
        XMLДок.ОткрытьЭлемент("Product");
        XMLДок.ЗаписатьЭлемент("Id", Строка(Выборка.Id));
        XMLДок.ЗаписатьЭлемент("Name", Выборка.Name);
        XMLДок.ЗаписатьЭлемент("Price", Формат(Выборка.Price, "ЧЦ=0; ЧД=2;"));
        XMLДок.ЗаписатьЭлемент("Quantity", Строка(Выборка.Quantity));
        XMLДок.ЗаписатьЭлемент("ImageUrl", Выборка.ImageUrl);
        XMLДок.ЗакрытьЭлемент();
    КонецЦикла;

    XMLДок.ЗакрытьЭлемент(); // Products
    XMLСтрока = XMLДок.Закрыть();

    // Сохраняем файл
    ЗаписатьТекст(ПутьКФайлу, XMLСтрока);
КонецПроцедуры

// Для публикации по HTTP:
// 1. Разместите обработку на веб-сервере 1С (публикация обработки).
// 2. В процедуре ОбработкаHTTPЗапроса отдавайте XML-строку как ответ:

Процедура ОбработкаHTTPЗапроса(HTTPЗапрос, HTTPОтвет)
    ПутьКФайлу = "C:\\Temp\\products.xml";
    СформироватьИВыгрузитьXML(ПутьКФайлу);
    XMLСтрока = ПрочитатьТекст(ПутьКФайлу);
    HTTPОтвет.УстановитьТелоИзСтроки(XMLСтрока, "text/xml; charset=utf-8");
    HTTPОтвет.КодСостояния = 200;
КонецПроцедуры

// Теперь файл будет доступен по HTTP, например:
// http://ваш_сервер_1с/yourmodule/products.xml

4. Советы
---------
- Проверьте права доступа на публикацию обработки.
- Для картинок используйте абсолютные URL (доступные с сайта).
- Если нужно — добавьте авторизацию (логин/пароль) на стороне 1С.

5. Как подключить к сайту
-------------------------
- На сайте реализуйте загрузку и парсинг XML по этому URL.
- Используйте данные для отображения каталога.

Если нужна помощь с конкретным кодом под вашу конфигурацию 1С — дайте знать!
